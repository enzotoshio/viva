# viva

Viva Real - Front-end Test

## Installation

```sh
$ npm i
$ npm i -g protractor
$ npm i -g bower
$ npm i -g grunt-cli
$ bower i
```

## Run

```sh
$ grunt serve
```

## Test

E2e tests - Protractor (Run each block in one separated bash in parallel):
```sh
$ grunt serve
```

```sh
$ webdriver-manager update
$ webdriver-manager start
```

```sh
$ protractor test/config.js
```
